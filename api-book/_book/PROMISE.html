接口约定

接口约定:

  • 如未特殊约定，大部分API都接受一个Object作为参数。
  • Object中可以指定succ，fail来接收接口返回结果。
  • 部分API中的succ和fail将作为'确认操作'或'取消'操作的回调函数，具体请查看相关API回调时机。
参数名 必选 类型 描述
succ false Function 接口调用成功回调
fail false Function 接口调用失败回调
